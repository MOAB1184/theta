# Scout Accelerator

A comprehensive platform for scouts to track their advancement, learn new skills, and connect for sign-offs. Includes admin panels for troop and rank management.

## Features

- **Rank Advancement Tracking**: Monitor progress through each scouting rank
- **Lesson Content**: Access comprehensive learning materials for each requirement
- **Digital Sign-off Requests**: Streamline the advancement process
- **Admin Dashboard**: Manage troops, ranks, and requirements
- **User Management**: Handle scout and leader accounts

## Getting Started

1. Clone the repository
2. Install dependencies with `npm install`
3. Set up your Supabase database
4. Configure environment variables
5. Run `npm run dev` to start the development server

## Environment Variables

Create a `.env.local` file with:
- `SUPABASE_URL`: Your Supabase project URL
- `SUPABASE_ANON_KEY`: Your Supabase anonymous key

## Deployment

The app is configured for Vercel deployment with proper SPA routing and build optimizations.
